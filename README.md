TaiwanTravelMaps
====

2015/12/05(土)~2015/12/07(月)に社員旅行で台湾に行った時のルートをLeafletを使って作成してみました。

## Description
* 1日目

  紅毛城⇒漁人埠頭⇒淡水老街⇒関渡媽祖宮⇒鼎泰豊(SOGO復興店)⇒士林市場
* 2日目
  
  故宮博物院⇒忠烈祠⇒大山茶芸教室⇒台北101⇒中正記念堂⇒龍山寺⇒
  
  中華民国総統府(車窓見学)⇒金龍藝品⇒梅子餐廳(南京店)
* 3日目
  
  九份⇒東楽台菜海鮮餐庁⇒昇恒昌免税店
